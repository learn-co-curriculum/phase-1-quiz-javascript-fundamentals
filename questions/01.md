# Fundamentals: Conditionals

Given the following code:

```javascript
const age = 6;
const isAdult = age >= 18;
isAdult; //=> ___
```

What will the value of `isAdult` be?

## Correct

false

> Correct!
> 
> If the value of `age` is less than 18, the value of `isAdult` is false.
